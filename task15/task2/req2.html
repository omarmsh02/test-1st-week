what is the differnce between HTML and CSS

1)HTML (HyperTextMarkup Language) is a markup language used to structure the content of a web page, while CSS (Cascading Style Sheets) is used to style and visually lay out the content on a web page. HTML provides the structure and content of a web page, while CSS provides the visual design.

Other differance between HTML

2)HTML is used to structure the content on the web page. CSS is used to add style to the content of a web page. HTML provides display information of various tags to the browser. CSS enhances that information by providing styling to those same HTML tags.
